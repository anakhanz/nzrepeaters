NZ Amateur Beacons, Digipeaters, Rpeaters and Sites

Data Sources

The information in the maps comes from the following sources:

  • RSM Smart
    • Frequency (the input frequency for repeaters is calculated from the output frequency)
    • Type
    • Site Name
    • Map Reference
    • Coordinates
    • Licence Number
    • Licencee
  • Callbook
    • Callsign
    • CTCSS
    • Branch
    • Trustees
    • Notes

Corrections

RSM Smart

For corrections to the data from the RSM you will need to get this corrected through the normal licence change process.

Call book

For corrections to data sources from the NZ call book please email zl2wal@nzart.org.nz including the licence number.
